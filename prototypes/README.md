# 匿名設計原型

這個資料夾是給外部設計協作者討論用的靜態原型，不載入正式題庫，也不顯示正式網頁標題。協作者可能是網頁設計師，也可能是一般美術人員、插畫師或擅長視覺風格的人。

## 使用方式

- 直接用瀏覽器打開 `prototypes/index.html`
- 主要修改 `prototypes/styles.css`
- 所有配色集中在 `:root` 與 `.prototype.theme-*`
- 真正要替換的品牌名、題目、插圖都以「保留位」標示

若只想拿去開會討論，可以從瀏覽器列印成 PDF，或用截圖工具輸出 PNG。HTML/CSS 仍保留作為可重新編輯的母稿。

給非網頁設計師時，建議提供 PDF 或 PNG，請對方直接在圖上標註色彩、插圖、字體、間距與情緒方向；等方向確定後，再由前端把設計整理回 HTML/CSS。

## 原型方向

1. `theme-table-card`：桌卡感，偏溫暖、穩定、適合多人聚會。
2. `theme-journal`：手帳感，偏可愛、親密、適合朋友與情侶。
3. `theme-event`：活動卡感，偏清爽、有主持感，適合未來付費題包或活動包。

## 保密原則

- 不放正式產品名稱。
- 不放正式題目內容。
- 不引用正式 `questions-data.js`、`source-questions-data.js` 或 `app.js`。
- 原型只討論版型、節奏、視覺語言與可編輯元件。
