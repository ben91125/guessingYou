# SOURCE 題庫候選資料

這個資料夾用來存放尚未正式入庫的候選題。這些題目不會被現有遊戲程式直接讀取，目的是讓產品方先審核、篩選、改寫，再決定是否放入正式題庫。

## 檔案結構

```text
SOURCE/
  README.md
  sources.md
  categories/
    01_life_habits.md
    02_interests_hobbies.md
    03_work_learning.md
    04_consumption_values.md
    05_relationships.md
    06_love_values.md
    07_life_values.md
```

## 題號規則

- `LH-001` 到 `LH-100`：生活習慣
- `IH-001` 到 `IH-100`：興趣嗜好
- `WL-001` 到 `WL-100`：工作與學習
- `CV-001` 到 `CV-100`：消費觀
- `RL-001` 到 `RL-100`：人際關係
- `LV-001` 到 `LV-100`：愛情觀
- `LF-001` 到 `LF-100`：人生觀
- `SI-001` 到 `SI-100`：自我認識
- `FG-001` 到 `FG-100`：家庭與成長
- `ES-001` 到 `ES-100`：情緒壓力
- `BC-001` 到 `BC-100`：界線與衝突

## 欄位

每題先用 Markdown 表格保存：

- 題號：穩定引用用，不代表正式題庫排序
- 建議模式：朋友、熟人、情侶，僅作初步建議
- 深度：低、中、高，方便篩選侵入性
- 候選題：目前的題目文案

## 使用原則

- 這些題目是候選素材，不是正式題庫
- 題目入庫前應由產品方審核語氣、界線、分類與重複度
- 參考來源只用於題型研究與靈感整理，不直接照抄商業題卡或文章題目
- 如果某題接近既有公開題目，入庫前應重新改寫
