# 匿名產品 UI Sample

這個資料夾是給 Sales 或內部展示用的匿名產品畫面 sample。

它和 `style-samples/` 不同：

- `style-samples/`：展示風格氣氛，可以不像真實產品。
- `product-samples/`：貼近實際 `index.html` 的 UI 層級與操作結構。

## 檔案

- `index1.html`：溫馨桌遊產品畫面
- `index2.html`：朋友派對產品畫面
- `index3.html`：情侶約會產品畫面
- `index4.html`：復古題卡產品畫面
- `styles.css`：產品 sample 的少量覆寫樣式

## 隱藏規則

- 不顯示正式產品名稱。
- 不放正式題目內容。
- 不載入正式題庫或正式遊戲程式。
- 保留真實 UI 層級：上方工具列、L1 關係、L2 題型、題號、玩家識別、評分、評語與紀錄。
